# Code of Conduct

This repository is maintained for research, documentation, and reproducible scientific workflows.

## Expected behavior

Contributors and collaborators are expected to:

- communicate respectfully and professionally
- give constructive, specific feedback
- document technical changes clearly
- respect project ownership, data-use restrictions, and publication timelines
- avoid committing sensitive, restricted, or unauthorized data

## Unacceptable behavior

Unacceptable behavior includes:

- harassment, intimidation, or personal attacks
- intentionally disruptive comments or pull requests
- sharing private, restricted, or unpublished data without approval
- misrepresenting authorship, contribution, or analysis provenance
- altering manuscript-linked workflows without clear documentation

## Research integrity

For repositories connected to posters, manuscripts, theses, reports, or public software releases, contributors should preserve reproducibility by documenting:

- input data sources
- analysis settings
- dependency changes
- output changes
- release versions
- known limitations

## Reporting concerns

Concerns should be reported to the repository maintainer, project lead, or PI according to the lab's internal communication process.

## Scope

This code of conduct applies to repository interactions, including issues, pull requests, code reviews, documentation edits, and project discussions.
