## Data archive checklist

Before finalizing a data availability statement, confirm:

- [ ] The repository URL is correct.
- [ ] The release tag is correct.
- [ ] The DOI resolves correctly.
- [ ] The dataset description matches the actual archived files.
- [ ] Raw versus processed data are described accurately.
- [ ] Restricted or excluded data are explained.
- [ ] The PI or project lead approved the public data statement.
- [ ] The statement matches the journal, thesis, or conference requirements.
